The Three Core Languages

At the heart of all computing lies the number system. While computers think in binary, we interact in decimal, and developers often use hexadecimal as a convenient shorthand. Each system has a specific role based on its strengths.

Decimal (Base-10)

Representing Whole Numbers

Integers are used for counting, indexing, and controlling programs. The method used to store them determines the range of available values. The modern standard, Two's Complement, is efficient because it has one representation for zero and simplifies CPU logic.

The Range Trade-Off: 8-Bit Integers

Using a bit to represent a sign (+) or (-) cuts the maximum positive value in half but opens up a range of negative values.

Handling Fractions & Vast Scales

For real numbers, science, and graphics, computers use a "floating point" system defined by the IEEE 754 standard. It breaks a number into a sign, an exponent (for scale), and a fraction (for precision), allowing for an enormous range of values.

Encoding Human Language

Text is represented by mapping each character, symbol, and emoji to a number (a "code point") via the universal Unicode standard. An encoding scheme like UTF-8 then translates this number into a sequence of binary bytes.
